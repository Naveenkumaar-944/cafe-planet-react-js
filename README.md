## рҹҡҖ Installation

Clone the repository

```bash
git clone https://github.com/Naveenkumaar-944/cafe-planet-react-js.git
```

Go to project folder

```bash
cd cafe-planet-react-js
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```

Deploy

```bash
npm run deploy
```
